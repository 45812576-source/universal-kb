# 竞标策略顾问

## Overview

帮助抖音运营公司项目总监在售前竞标阶段，通过结构化引导分析项目信息，输出差异化的**提案策略建议书**。策略建议书将作为后续方案生成（proposal-generator）的输入。

核心原则：**先诊断，再开方** —— 不同项目类型、品牌阶段、预算量级、竞标场景下，方案的侧重点、报价策略、差异化锚点完全不同。

## When to Use

- 收到品牌客户的招标邀请/brief，需要制定提案策略
- 需要在竞标前分析竞争格局、确定差异化定位
- 需要决定方案的侧重方向和报价策略

## 引导流程

按以下三层维度逐步引导用户输入信息，**每次只问一个维度**，使用多选题优先。每个维度的回答都会影响最终策略输出。

### 第一层：项目基本面

**1. 项目类型**（可组合）

| 类型 | 说明 | 方案侧重 |
|------|------|---------|
| 短视频素材投放 | 千川/信息流为主的效果广告 | 投放策略分层、素材方法论、人群定向、成本控制模型 |
| 达人种草/内容营销 | KOL/KOC矩阵种草 | 达人矩阵规划、内容方向拆解、种草转化链路 |
| 账号代运营 | 品牌官方号运营 | 账号定位、人设策略、内容日历、涨粉路径 |
| 整合营销/Campaign | 大型品牌活动 | Big idea、传播波次、跨触点联动 |
| 组合项目 | 以上多种组合 | 各模块协同逻辑、预算分配优先级 |

**2. 投放渠道**（可多选）

| 渠道 | 平台工具 | 策略特点 |
|------|---------|---------|
| 抖音 | 千川 | 短视频引流、直播投流、商品卡推广 |
| 抖音 | 巨量AD | 品牌广告、开屏、搜索 |
| 京东 | 京准通/JDS | **CID引流**为主，站外投放引流至京东成交 |
| 阿里 | 阿里妈妈/UDS | **CID引流**为主，站外投放引流至天猫成交 |
| 腾讯 | 腾讯广告 | 朋友圈、视频号、小程序链路 |
| 快手 | 磁力金牛 | 短视频+直播投放 |
| 小红书 | 聚光/薯条 | 种草笔记投流、搜索卡位 |

注意：京东和阿里侧重CID引流模式（站外短视频投放→CID追踪→电商平台成交），不是站内投放。需要在方案中体现跨平台归因能力。

**3. 品牌阶段**

- 新品牌/新品上市 → 冷启动方案、人群破圈、声量打法
- 成熟品牌日常运营 → 效率提升、精细化运营、数据驱动优化
- 品牌升级/转型 → 新人群/新赛道洞察
- 新品上市（成熟品牌的新品线） → 老客触达 + 新客破圈组合

**4. 预算量级**

| 量级 | 短视频投放预算 | 方案特点 |
|------|-------------|---------|
| 头部 | 年5000万+ | 战略高度、资源壁垒、独家权益、多渠道协同 |
| 中腰部 | 年1000万级 | 组合拳策略、资源整合、效率优先 |
| 白牌/抖品牌 | 年500万以下 | 单一链路打透、小而精、快速见效 |

直播预算通常为短视频投放的数倍，需单独沟通。

**5. 行业品类**

引导用户指定行业品类，加载对应行业知识附件（`industries/` 目录）。如无对应附件，引导用户提供行业基础信息。

### 第二层：竞标情境

**6. 竞标场景**

- 正式比稿竞标 → 严格对标RFP每条要求，逐项响应评分标准
- 定向邀请提案（最常见） → 方案质量和团队感觉决胜，突出差异化
- 商务关系驱动 → 方案作为合作评估依据，重点建立信任

**7. 客户决策风格**

- 数据驱动型（电商品牌居多） → 方案必须有详细数据模型、ROI测算、漏斗拆解
- 创意驱动型（美妆时尚居多） → 亮眼创意demo、视觉参考、内容样板
- 老板拍板型（中小品牌） → 简洁有冲击力，核心卖点3页讲清楚
- 采购流程型（大集团） → 对标RFP，评分项逐一响应

**8. 竞争格局**

引导用户思考：
- 已知竞标对手是谁？他们的强项是什么？
- 客户与现有服务商的关系如何？是换还是加？
- 我们的差异化锚点在哪？（别人强执行我们强策略，别人强策略我们强数据）

**9. 客户核心关注点**

- 费比（服务费占比/性价比） → 当前市场竞争下越来越重要
- ROI/效果保障
- 内容质量/创意水平
- 团队稳定性/服务响应
- 行业经验/同品类案例

### 第三层：甲方资源摸底 & 我方定位

**10. 甲方资源摸底**

此维度有**双重用途**：
1. **策略阶段**：内部判断方案可执行性，决定哪些策略方向可行
2. **提案输出**：作为方案的正式章节，向客户反向提出资源配合要求，体现专业度和落地思维

| 摸底维度 | 影响范围 |
|---------|---------|
| 素材储备 | 现有素材量、质量、能否二次剪辑 → 素材产能规划和成本 |
| 达人合作深度 | 长期合作达人、历史种草数据 → 达人策略是新建还是优化 |
| 工厂/溯源资源 | 能否拍工厂、原料产地 → 内容方向是否走"溯源背书"路线 |
| 库存备货能力 | 大促期间供应链能否跟上 → 投放节奏是否敢放量 |
| 价格机制 | 控价体系、渠道比价、促销力度空间 → 转化策略和利益点设计 |
| 品牌资产 | 代言人、IP授权、品牌故事素材 → 是否有高势能素材可用 |
| 直播排期 | 品牌自播节奏、大场直播计划、主播档期 → 短视频投放与直播的协同节奏 |

**11. 我方优势盘点**

- 核心打法定位：投放能力 / 内容创意 / 数据驱动 / 行业深耕
- 同品类案例及数据
- 可调用资源：达人、平台关系、独家权益
- 团队配置

**12. 报价策略**

综合考虑以下因素给出报价建议：
- 公司成本毛利底线
- 市场均值/行业费比水平
- 客户战略价值（标杆案例、品牌背书）
- 其他业务cross-sell空间
- 是否需要阶梯式KPI对赌

## 策略输出物

完成引导后，输出**提案策略建议书**，包含以下模块：

### 1. 差异化定位
- 我方核心卖点 vs 竞争对手的错位策略
- 本次提案的"一句话定位"

### 2. 方案侧重建议
- 各章节权重分配（重点展开 / 标准 / 简写）
- 哪些内容是"赢单点"，需要重点投入

### 3. 内容策略方向
- 建议的创意大方向
- 内容调性和风格建议
- 素材类型组合建议

### 4. 多渠道协同策略
- 各渠道的角色定位（种草/收割/品宣）
- 跨渠道链路设计（如：小红书种草→抖音短视频放量→CID引流至京东/天猫成交）
- 预算分配建议比例

### 5. 报价策略
- 定价思路和报价结构
- 服务分层方案（基础+增值）
- 对赌/激励机制建议（如适用）

### 6. 风险预判与应对
- 客户可能的质疑点 + 预备回应话术
- 方案执行风险及备选方案

### 7. 提案演示建议
- 演示重点和节奏
- Q&A预判（基于客户决策风格和核心关注点）

### 8. 甲方配合事项清单
- 策略落地所需的甲方资源和配合项
- 需要在提案前/提案中确认的关键问题

## 输出完成后

策略建议书确认后，引导用户进入 **bidding-proposal** skill 生成完整竞标方案。
