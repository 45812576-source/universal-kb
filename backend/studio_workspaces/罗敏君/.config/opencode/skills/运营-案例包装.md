```markdown
# 案例包装 Skill（单页精简版）

## Skill 标识
- **名称**：case-packaging-onepage
- **别名**：单页案例、案例卡片、极简案例包装
- **功能**：生成结构化案例文案（含策略简述）+ 可选的单页PPT设计稿
- **输出结构**：项目概况 → 策略简述（短篇） → 关键成果 → 亮点复盘 → 可复用经验

---

## 初始化指令（/init）

请发送以下信息，**带*为必填**：

```
【基础信息】
*品牌名称：
*所属行业：3C/美妆/服饰/教育/食品/其他
*投放平台：UDS/千川/抖音CID/小红书/快手/JDS
*投放周期：如2025年4月
*核心产品：如充电器、充电宝等多款新品

【核心数据】*至少3项
*ROI：
*点击率/互动率：
*转化率：
*曝光量/GMV：
*获客成本：

【策略关键词】（选填）
如：阶梯出价、多SKU测试、素材矩阵、智能放量等

【项目亮点】（选填）
- 数据突破点：
- 方法论创新：

【风格偏好】商务/科技/极简/电商
【是否需要PPT】是/否（默认否）
```

## 标准输出结构

\```markdown

**周期**：【时间】  
**平台**：【平台】  
**产品**：【核心产品】  

**核心挑战**：  
- 【客户痛点1】
- 【客户痛点2】

## 二、投放策略简述
【一句话概括核心策略，如：采用阶梯出价+多SKU赛马机制，冷启动期建立模型，测试期筛选爆款，放量期智能控本；素材端剧情+场景+促销三轨并行，覆盖用户全决策链路。】

| 点击率 | 【X%】 | 【高于大盘XX%】 |

## 快捷调用模板

### 极简模式（推荐）

```
包装案例：【图拉斯】在【UDS】投放【充电器/充电宝】，ROI【7.13】，点击【3%】，转化【5%】，策略是【阶梯出价+多SKU测试】，【需要PPT】

```
品牌：图拉斯
行业：3C数码
平台：UDS
周期：2025年4月
产品：充电器、充电宝多款新品

## 平台策略速查（一句话适配）

| 平台 | 策略简述模板 | 关键数据组合 |
|-----|------------|-------------|
| **UDS** | 阶梯出价+多SKU测试，冷启动建模→测试筛选→放量控本 | ROI+点击率+转化率 |
| **千川** | 智能放量+短视频引流+直播间收割，分时段出价优化 | ROI+GPM+成交单量 |
| **抖音CID** | 精准归因+电商闭环，素材A/B测试+人群包迭代 | ROI+转化率+获客成本 |
| **小红书** | KFS种草（KOL+Feeds+Search），内容矩阵建立品牌心智 | 互动成本+CPE+搜索提升 |
| **快手** | 老铁经济+私域沉淀，真人出镜+福利诱导+粉丝运营 | 私域转化+复购率+粉丝成本 |

## 执行规则

1. **策略篇幅**："投放策略简述"严格控制在1-2句话，50字以内，仅作过渡
2. **保留模块**：必须包含【关键成果（表格）】【亮点复盘（3点）】【可复用经验（方法论）】
3. **PPT触发**：默认仅输出文案；用户明确说"要PPT"/"生成PPT"/选择"是"时，追加单页PPT设计稿
4. **数据缺失**：若缺少ROI或平台信息，先追问再输出；其他数据可先用"XX"占位
5. **一页原则**：PPT设计稿严格遵循单页结构，禁止分页
6. **即拿即用**：文案可直接复制到PPT备注页，设计稿可直接交给设计师执行
