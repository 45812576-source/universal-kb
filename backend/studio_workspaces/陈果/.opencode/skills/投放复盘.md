# 投放复盘

## 功能说明

该技能用于分析广告投放效果，提供策略优化建议，帮助用户进行投放复盘。支持多个主流投放平台的数据分析和策略建议。

## 投放平台知识

### 阿里巴巴UDS
- **全称**：Unified Data System
- **核心功能**：阿里巴巴生态内的广告投放数据管理系统
- **关键指标**：点击量、展现量、点击率(CTR)、转化率(CVR)、客单价、ROI
- **优化方向**：人群定向、创意优化、出价策略、投放时间

### 字节CID
- **全称**：Click ID
- **核心功能**：字节跳动广告投放的转化追踪系统
- **关键指标**：转化量、转化成本、转化路径、归因模型
- **优化方向**：转化追踪设置、素材质量、投放时段、定向人群

### 千川
- **全称**：巨量千川
- **核心功能**：字节跳动电商广告投放平台
- **关键指标**：GMV、ROI、客单价、转化率、点击率
- **优化方向**：商品选择、创意素材、投放策略、人群定向

### 京东JDS
- **全称**：京东直投系统
- **核心功能**：京东平台内的广告投放系统
- **关键指标**：点击量、转化量、ROI、客单价、广告费用
- **优化方向**：商品详情页优化、关键词选择、出价策略、投放时间

### 小红书
- **核心功能**：内容营销和广告投放平台
- **关键指标**：曝光量、互动量、点击率、转化率、笔记质量
- **优化方向**：内容质量、KOL选择、话题策划、投放时机

### 腾讯广告
- **核心功能**：腾讯生态内的广告投放系统
- **关键指标**：曝光量、点击率、转化率、社交互动、ROI
- **优化方向**：人群定向、创意形式、投放位置、预算分配

### 快手
- **核心功能**：短视频平台广告投放系统
- **关键指标**：播放量、互动率、点击率、转化率、ROI
- **优化方向**：视频质量、内容创意、投放时段、定向人群

## 使用方法

1. **提供投放数据**：用户需要提供投放平台、投放时间范围、投放数据（如点击量、展现量、花费等）
2. **选择分析维度**：用户可以选择按平台、按时间、按创意等维度进行分析
3. **获取分析结果**：系统会根据提供的数据和平台知识进行分析，生成投放效果报告
4. **接收优化建议**：系统会基于分析结果提供具体的投放策略优化建议

## 分析流程

1. **数据收集**：收集用户提供的投放数据
2. **数据清洗**：处理数据中的异常值和缺失值
3. **指标计算**：计算各项关键指标（如CTR、CVR、ROI等）
4. **效果分析**：分析投放效果，识别优势和不足
5. **策略优化**：基于分析结果和平台知识，提供具体的优化建议
6. **报告生成**：生成详细的投放复盘报告

### 执行摘要
- **核心指标**：点击率、转化率、ROI等关键指标的表现
- **趋势分析**：投放效果的变化趋势
- **优势劣势**：投放策略的优势和需要改进的地方
- **行业对比**：与行业平均水平的对比

### 素材优化建议

#### 执行摘要
- **策略调整**：需要调整的投放策略
- **素材优化**：需要改进的素材内容
- **预算分配**：预算的调整建议
- **时间安排**：投放时间的优化建议

1. **视频素材优化**：
   - **开头优化**：前3秒必须有强钩子，如问题引导、冲突场景、惊喜元素
   - **内容结构**：采用「问题-解决方案-行动号召」结构
   - **长度控制**：短视频控制在15-30秒，长视频控制在1-2分钟
   - **视觉效果**：使用明亮色彩、动态元素、字幕增强可读性
   - **声音设计**：添加背景音乐，确保音量适中，关键信息有语音强调

2. **图片素材优化**：
   - **视觉焦点**：主体清晰，突出核心卖点
   - **色彩搭配**：使用品牌主色调，对比鲜明
   - **文字排版**：字体大小适中，重点信息突出
   - **构图技巧**：黄金分割法，留白合理
   - **格式优化**：确保图片清晰，文件大小适中

3. **文案素材优化**：
   - **标题优化**：使用疑问句、数字、惊叹号等吸引注意力
   - **卖点提炼**：突出产品核心优势，解决用户痛点
   - **语言风格**：符合目标人群语言习惯，避免专业术语
   - **行动号召**：明确引导用户行动，如「立即购买」、「点击了解」
   - **A/B测试**：测试不同文案效果，选择最优版本

4. **落地页优化**：
   - **加载速度**：确保页面加载时间小于3秒
   - **首屏设计**：首屏必须包含核心信息和行动按钮
   - **内容结构**：逻辑清晰，层次分明
   - **表单设计**：简化表单填写，减少用户流失
   - **移动端适配**：确保在不同设备上显示正常

```
# 字节CID投放复盘报告

## 基本数据
- 投放时间：2024年1月1日-1月7日
- 总花费：10000元
- 点击量：5000次
- 转化量：200个
- 客单价：200元

## 关键指标
- 点击率(CTR)：1.2%（行业平均：1.0%）
- 转化率(CVR)：4.0%（行业平均：3.5%）
- 转化成本：50元
- ROI：4.0

## 效果分析
- 点击率高于行业平均，说明创意质量较好
- 转化率略高于行业平均，有提升空间
- ROI达到4.0，效果良好

## 优化建议
1. **创意优化**：
   - 调整视频开头3秒内容，增加钩子元素（如问题引导、冲突场景）
   - 测试3-5个不同风格的创意素材，保留CTR前20%的素材
   - 优化视频结尾CTA，明确引导用户行动（如点击链接、下单）

2. **定向优化**：
   - 排除近7天已转化用户，避免重复投放
   - 增加高转化人群的相似人群扩展，扩展比例设置为15%
   - 调整年龄定向，将25-34岁人群预算占比提升至60%

3. **出价优化**：
   - 对转化成本低于目标的计划，逐步提高出价5-10%
   - 对转化成本高于目标30%的计划，降低出价15%
   - 为高ROI计划设置预算上限，确保充分消耗

4. **投放时间**：
   - 将60%预算分配到转化高峰时段（20:00-22:00）
   - 测试早间（8:00-10:00）和午间（12:00-14:00）时段效果
   - 暂停凌晨0:00-6:00的投放，提高预算利用效率

5. **账户调整**：
   - 新建2个测试计划，分别测试不同创意方向
   - 对表现差的计划（CTR<0.5%）进行暂停或重建
   - 调整账户结构，按产品类别或创意类型分组管理计划

6. **数据监控**：
   - 设置转化成本预警阈值，超过目标20%时及时调整
   - 每日监控素材效果，及时替换低表现素材
   - 每周分析账户数据，总结优化效果并调整策略
```
